Detailed Description

A simple lexical analyzer.

Tokenizer parses string values and recognizes floating point values and arithmetic operators.

Current token type, operator, or literal.

Returns:

TokenValue Tokenizer::crtTokenValue ( )

The value associated with the current token.

Returns:
a TokenValue union representing the value of the token. If the current token is a literal, then the literalVal field equals the value of the literal. If the token is an operator, then the opVal field identifies the operation
void Tokenizer::nextToken ( )

Reads the next token from the string.

The next token from the string is parsed. Deetails about the new token are made available by the crtToke(Type/Value) functions. If there are no more tokens to be processed from the string, then crtTokenType returns the Tokens::THEEND value and crtTokenValue returns a undefined value.
